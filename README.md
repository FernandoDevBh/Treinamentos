# Treinamentos
Solução para implementar os requisitos para o Treinamento de Teste Unitários

1. Parar executar essa solution é necessário que esteja instalado em sua máquina a versão mais recente do dotnet core (foi desenvolvido na versão 3.1.402)
2. Baixar a solution em um diretório de sua preferência.
3. Navegar para o diretório /Treinamentos e executar o comando: `dotnet restore`
4. Ainda no diretório Treinamentos excutar o comando: `dotnet build`
5. Navegar para o diretório Treinamentos.View e executar o comando `dotnet run`
